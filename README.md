# Licenta Handrau Codin

This project is a web application for price monitoring and order management.

## Structure

-   `frontend`: Contains the React frontend application.
-   `licenta-backend`: Contains the Python Flask backend, including a machine learning model for price prediction.

## Setup

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd licenta-backend
# It is recommended to use a virtual environment
pip install -r requirements.txt
flask run
```
